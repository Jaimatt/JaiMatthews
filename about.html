General Information

The website you are now looking at was created by Jai Matthews and was built from the ground up using HTML, CSS & JavaScript on Microsoft's VS Code. It is currently being hosted using GitHub. This website serves as a digital portfolio for Jai Matthews. It contains a variety of projects, creations, and interests including other websites created for school assignments, Illustrations, Mystery Nights, and more. Most notably is the illustrations page containing all of my artworks I've created on adobe illustrator. Have a look at that page to access more information about each illustration. Please enjoy the site.
