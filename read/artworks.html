Artworks

Jai Matthews

Since late 2020 I've been creating a number of artworks — often vector-based landscapes — and posting them to my Instagram.

The 'artworks' page on my website uses the Instagram Basic Display API to fetch the posts from Instagram to display on my website. With the API key, I have access to a JSON file containing the images ready to be displayed. Then, the page constructs the HTML from this information and displays it.

JSON received from Instagram API.

The page is equipped to handle and display images of different aspect ratios and can display all items of 'carousel' images.
